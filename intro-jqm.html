What's jQuery Mobile?

  • Builds on jQuery core library
  • Free and open-source
  • Framework for building mobile sites and apps using web code
    • Not just a JS library, but a UI library
    • Pre-built components, styles / themes and actions
    • HTML5, CSS3 and JS
    • Full support for iOS, Android, Blackberry, Windows phone, WebOS
    • Graded support for many more

Why jQuery Mobile?

  • Same reason as for any library / plug-in / gem — leverage existing well–vetted codebase to quickly get things done
  • Mobile browsers (Android and iOS) are more likely to support bleeding edge effects
  • Quick and dirty mobile-focused site when full site is old–fashioned / Flash–based
  • Together with other apps and frameworks, make mobile apps!

Examples

  • Mobile Site: Stanford.edu
  • Mobile site: Walt Disney World
  • App: Kapicu

Apigee API

Load Data

function loadBooks () {
  my_books.fetch(
    function () {
      $('#books-list').empty();
        while ( my_books.hasNextEntity() ) {
          var current_book = my_books.getNextEntity();
          // Output the book on the page
          $('#books-list').append('<li><h3<'+current_book.get('title')+
                    '</h3><p>'+current_book.get('author')+'</p></li>');
        }
    }
  );
}

Apigee API

Create Data

function createBook() {
  new_book = { "title":$("#title-field").val(),
               "author":$("#author-field").val() };
  my_books.addEntity(new_book, function (error, response) {
      if (error) {
          alert('Write failed.');
      } else {
          $("#title-field").val(""), $("#author-field").val("");
          loadBooks();
          history.back();
      }
  });
}

PhoneGap & Xcode

Port to iOS

  1. Open Terminal, and navigate to ~phonegap/lib/ios/bin
  2. Type ./create [path-to-your-web-app]
  3. In Terminal type open ~/[path-to-your-web-app] or just navigate to it in the Finder
  4. Go to the www folder in the newly created project folder
  5. Delete everything except cordova.js
  6. Copy your web apps's indes, js, css and image assets here
  7. Now you can open that new project in Xcode, and view it in the Xcode iOS simulator. Testing on an iOS device requires an Apple developer account ($99/year).
